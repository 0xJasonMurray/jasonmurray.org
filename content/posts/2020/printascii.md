Python script to dump the ASCII table:

```
#!/usr/bin/env python3

print("|{:^9}|{:^10}|{:^10}|".format("ASCII", "Decimal", "Hex"))
for c in range(256):
    print("|{:^10}|{:^10}|{:^10}|".format(str(chr(c)), c, hex(c)))
```
