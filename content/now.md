# Work

- Working for [Washington University in St. Louis](https://wustl.edu/) as a Network Architect:
  - Creating a 3 year networking strategy that takes into account the full stack of networking services. We no longer design networks in a bubble. A holistic design which includes networking, security, desktop tooling, servers, segmentation, and operational management. I am bringing  architects together from multiple disciplines to build a next generation network that works for everyone.
  - Four big projects: 1) Installing 100G links and routers to Equinix to facilitate connections to the big cloud providers, content providers, and the Internet; 2) Evaluating a software defined architecture for our primary client production network; 4) Evaluating software define architecture for all our data center locations; 4) Finalizing a NIST-800-171 network design.
  - Learning how LISP, VXLAN, anycast Gateways, etc work together in a modern network design. While network controllers abstract many of these details away from the engineers, I am building a lab to manually configure these components to gain operational experience with the protocols.
  - Working to support our medical, research, and students so they can focus on stu- dies, treating patients, and finding solutions for the COVID-19 crisis.
